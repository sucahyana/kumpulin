API Spesifikasi Aplikasi Acara

Ini adalah API untuk Aplikasi Acara yang memungkinkan pengguna untuk membuat dan mengelola acara, melihat profil pengguna, berinteraksi dengan peserta acara, dan melakukan pencarian pengguna atau acara.
